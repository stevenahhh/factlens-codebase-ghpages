WHY US04 / 11

차별점

단순 AI 요약이 아니라, 주장 단위 검증 + 근거 추적 + 판정 구조화.

01

주장 단위 분리 검증

문서 전체 요약이 아니라, 검증 가능한 주장만 뽑아 각각 독립 판정.

02

3단계 Fallback 근거 수집

KB → 인터넷 실시간 검색 → 내장 자료 순으로 탐색. 근거 없이 판정하지 않음.

03

4가지 고정 라벨

근거 있음 · 충돌 · 근거 부족 · 과장 표현. 정형화된 결과로 후처리에 바로 사용 가능.

04

검증 리포트 출력

판정 라벨 + 근거 출처 + 수정 제안을 묶어 반환. 바로 활용 가능한 형태.

05

서버리스 AWS 풀스택

Lambda + Bedrock KB 기반. 파일 업로드부터 판정까지 AWS 단일 파이프라인으로 운영.

06

자율성 준비된 구조

규칙 기반 라우팅 + AgentCore 연계 설계. 현재 동작하며, 확장도 가능한 상태.
